放弃前端，重新开始

发表于 2018-01-03 | 分类于 有关生活 | | 阅读次数

     计划在17年底写完的，拖延症一犯就到了18年。本着新的一年得有个好的开头的执念（还在不要脸的刷剧），我终于打开了电脑码（huan）字（zhai）。

    人的想法随着时间的推移总是在变，这是毋庸置疑的。就拿最近的我来说，天天在寝室唠叨着要回家卧床休养（撕脱性骨折），好让室友“郭神棍”在寝室“享受”个人生活（这小伙子可害怕一个人了）。然而前前后后立了不下十次的flag，最终都以啪啪啪打脸和“郭神棍”的嘲笑告终。立flag是真心的，但每次我都能用各种奇葩理由劝服自己留在学校。

    实习以来，要不要继续做前端的想法几次更迭，中间一度十（yong）分（ren）困（zi）扰（rao），和高中同学聊过之后又决定再坚持，几经反复，最后以秋招放弃前端告终。
    放弃的理由很多，最任性也是最主要的是不喜欢、不开心。做与不做的想法一直在变，不喜欢写代码从未改变。一定会有人说，大多数人做的工作都不是自己所喜欢的。的确如此，在“干一行，爱一行”还是“爱一行，干一行”这个辩题中，“XX%人做的工作都不是自己喜欢的工作”一定会成为前者的论据。仔细思考这个辩题，社会这个大环境一定是推崇“干一行，爱一行”，而“爱一行，干一行”更多的是个人理想情怀，是更高层次的追求。所以前者应该是打“现实主义”牌，后者应该打“理想主义”牌。以前，为了寻求高薪选择前端，是现实所迫；现在，经历了这所谓的现实，这次我站理想主义，即使是我只是找到了自己不想做什么。感谢教育提供给了我选择的多样性，让我抛开前端并非无路可走。
    所有码农都知道，做开发只有去互联网公司才有前途；所有码农也都知道，去互联网公司做开发加班是常态。所以大多数开发乃至产品的生活状态是，周一到周五，早上9点多起来，工作到晚上10点，回家洗洗睡。不开心有木有？我有，暗无天日的感觉。朝十晚十，好与不好，见仁见智。“一千个读者就有一千个哈姆雷特”，而我不喜欢这种状态！！！
    前面提到教育，不得不再说说学历这个问题。我知道我要说的观点在知乎上肯定会受到许多人的口诛笔伐，我曾经也写过一篇关于读研与前端的博客，核心观点是读研与做前端并不冲突，读研可以得到除工作经验之外的很多东西。之前的观点是一个万能解答，相当于问“A和B哪个好”，你回答说“没有好坏之分，对有些人来说A好，另外一些人B好”。抛开中庸回答，我认为，做前端不需要读研，做前端不需要读研，做前端不需要读研，做开发重要的是技术能力，与学历无关，所以你读研获得的其他方面的成长远不如技术水平的提升重要，而读三年研的技术水平的成长还不如工作三月。总而言之，硕士做前端是一个性价比极低的选择。除此之外，对于硕士，还有些其它的困扰，比如带你的人比你还小，比如上级贬低研究生，比如受不了身边都是本科生，这些问题都是庸人自扰，受得了继续受，受不了去找受得了的工作。
    再来说说就业选择的另一个因素——城市。这个因素看实际上是结果而非原因，选择了前端几乎就相当于选择了一线城市，好的公司都在一线城市，如果不做前端于我而言就相当于放弃了一线城市。一线城市给我们提供了良好的就业环境与娱乐消费环境，但同时也给我们带来了巨大的生活压力。身边不少人直接留在了二线城市，不少人是抱着累积工作经验的心态去一线城市，想在一线城市扎根的少之又少。自始至终没有想过在一线城市扎根的我选择二线城市是一个毫不费力的选择。

    说了一堆放弃前端的理（jie）由（kou），最终成功的达到了欺骗自己的效果。真心为我自己感到骄（can）傲（kui），以后可以佛系的生活啦。从自黑中回到正题，放弃前端的根源其实是来自价值观、人生观的改变，当觉得金钱没有身体、快乐、家庭重要时，放弃前端也就无需犹豫。同时我也需要告诫未来的自己，选择轻松无可厚非，但是要一直保持学习的状态，不能让自己因为没有能力做其他工作而抱着一个稳定而没有前途的铁饭碗。自勉之。
