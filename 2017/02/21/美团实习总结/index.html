美团实习总结

发表于 2017-02-21 | 分类于 有关工作 | | 阅读次数

前言

从2016年10月31号入职到年前返校，我在美团实习了整整12周。如果用一句话来描述这次实习经历，用“睁眼看世界”来形容那是再贴切不过了。回想自己从刚进公司时拿着Mac不知如何找文件，到现在对大公司的前端开发流程有了比较全面的认知、知识面得到拓展、实践能力也提高些许，不得不说这三个月的收获超过预期，这里真的是要感谢美团、远哥、琪哥还有团队的小伙伴们。
有所得当然也有所失，学校的一切我都短暂的失去了，出来实习之前对实习有这强烈的心理需求，在实习的过程中，不断成长，不断调节心理预期，回学校的心理需求却是不断增强。
因此，我有必要好好总结过去这三个月的得与失，回首过去，计划将来。我将从工作、学习、生活、思想四个方面进行总结，从而决定未来这一学期的计划。

工作

三个月的工作做的东西倒不是很多，但是涉及到的知识面却比较广泛。此次工作虽然和自己想象中的不一样，但是却有别样的滋味。

  1. 最开始的两周是一个熟悉脚手架和开发流程的过程，尔后开始慢慢的接触红包业务，从一点点小的修改开始，慢慢熟悉node层，服务器端渲染，前端渲染。一步步的积累经验之后，开始自己做一个新需求，比如红包项目中的集点活动。从中我也尽量避免使用UI框架，给自己创造机会多手写CSS。比较闲的时候也会自己尝试写一点小东西，比如日期格式化函数、React联动选择的组件。利用React完成了一个新需求（广告投放管理）算是一件比较高兴的事情，实习期间花了比较多的时间来看React全家桶，虽然没有找到机会应用Redux，但是能用实践React已经感到很高兴了。
  2. 在学校时，臆想的前端开发应该从切图开始，到交互到逻辑处理，从css到js都要一手抓，但是在我们团队，大家似乎都比较重视js、node这一块的内容，css的编写较少。我一直感觉真正细究css，那花的时间比写逻辑花的时间都会多，所以侧重js逻辑这块倒是出乎意料的对我胃口。
  3. 写前端比较好的一点是，你做的东西你立刻就可以看到成果，你可以看到可视化的界面，当你看到你代码一点点的累积，你的作品一点点的呈现在你的面前，你可以感受到你的付出。我最初开始写PHP，做各种逻辑处理，但是没有一个合适的界面可以量化你的成果，会让人怀疑我到底做了多少工作，我做的工作到底有没有用。包括你做出的东西给别人看，别人看到的就是你这个作品巨丑，完全不会关注你做这个东西背后的逻辑。相信很多写后端的人被别人评价过，能不能把界面再做好看一点（这番话对后端来说真是巨大的打击）。
  4. 不知道是我个人的问题还是本是如此，在熟悉团队的开发流程之后，不断重复又重复的写代码竟然让我觉得有一点无趣，如果不应用新的技术栈，实在是没有继续写代码的动力。

学习

工作也是一个学习的过程，但是工作之余的学习更为重要。工作日的学习时间比较零碎，如何利用周末时间很关键。

  1. 时间。最开始的一个月是“睁眼看世界”的一个月，node、ES6、模块化、打包工具、Express、KOA以及各种前端MV*框架，各种技术让我仿佛发现了新大陆一般，直接把我从旧时代带到了新世界。发现新大陆那也只是发现而已，重要的还是如何吸收这些知识。所以最开始的一个月，周末的时间几乎都在家学习这些新知识。但是到越到后面就越感觉到累，周末花的时间也越来越少。
  2. 效率。在公司的学习效率低，毕竟是工作时间，自己学习毕竟不好；周末学习效率也不高，学习氛围直接影响学习效率。
  3. 计划。计划真的很重要、很重要、很重要。前一天睡觉之前，用软件计划一下明天该做的事情，状态不好的时候可以利用番茄时间来学习。
  4. 输出。这也是实习的一大收获，以前在学习新知识的时候，仅仅只有输入而没有输出，学了不用也不思考，一段时间就会忘记。因此，在今后的学习中，有应用场景一定要应用一番，找不到应用场景也应该做个总结，可以是做一个分享，也可以是写一篇博客。

生活

来北京的时候就有同学对我说，“来北京干什么，留在学校多好！”，三个月过去了，以后我也会将这句话送给其他人。对自己，我只想说一句话：“好好珍惜在学校的时光！”

思想

实习后的思想真的过于反转和揪心。从开始坚定的走技术这条路，到想继续求学，到认清现实，继续走前端这条路，一步步都是一个彻彻底底的否定以前的大转变。
回想自己读研究生的初心是那么的简单，就是好好做技术，然后进BAT。但是处在不同的阶段，追求就不一样了。出来实习后，对工作和上班后的生活也有了初体验，由此，对于职业选择，想法也出现了几次转变。第一次思想的转变出现在实习两个月后，那时只觉得前端的格局太小，这种生活也不是自己想要的，因此，一度近乎决定回学校去做数据分析和数据挖掘方向（刚进学校没有选这个方向就是因为已经学了一段时间的前端，舍不得放下），最后，在和其他同学聊了一番之后才决定再坚持一下，也许我会看到不同的东西；第二次思想转变是实习快三个月的时候，这次只是觉得我比较喜欢学校图书馆的这种环境，因此可以考虑一下回去读个博，但太多的客观条件不允许，这个想法也只能是想想罢了；最后一次思想的转变是在回学校之后，想了很多，也想的很远：的确，学校的生活是我想要的，但是现实是现阶段前端才是最适合我的方向，先找一个大的平台去见识更多的东西，也许我会看到感兴趣的东西，工作一两年后，而如果那时我还是觉得想留在学校，那么就辞职踏踏实实的考个博或者去国外读个双硕士。

总结

总而言之，此次实习，也许会成为我研究生阶段最重要的一段经历，此次对工作、生活和学习有的感悟已经直接影响到了我对未来的决策。同时，我也了解了自己的能力，坚定了走前端的信念，对下一阶段的学习也有了明确的目标。
实习的收获很大，但我最终还是决定离开。
当下，我更明白我需要的是什么，如果象牙塔是我最后的天堂，那么就让我在天堂里慢慢的走出来。
