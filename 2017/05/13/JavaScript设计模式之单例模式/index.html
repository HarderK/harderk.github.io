究其缘由，主要还是JavaScript不是一门面向对象的语言，而文章开头提到的定义都是针对面向对象的语言的。
在JavaScript中，主角还是函数，所以扩展到函数中，我们就应该这样理解，单例模式就是保证函数主体操作在重复执行的时候只执行一次。
